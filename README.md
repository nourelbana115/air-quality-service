# Air Quality
Node.js, Nest.js web server to fetch air quality from IQAIR API, in this project i demonstrate how to integrate with third party using axios in nest.js, create a cron job runs every minute with nest.js schedule.

### Dependencies  
-   [Node.js](https://nodejs.org/en/)
- [Nest](https://github.com/nestjs/nest)
- [Mongoose](https://mongoosejs.com/)

## Running the app in production mode
Note: please setup environment variables file so the application work properly i have pushed my env file so you can test it easy

- cd to 'Air_Quality' directory
- import ```Air Quality.postman_collection.json``` to postman
## Running the Node App in development mode

- cd to ```'currency-exchanger/backend'``` directory
- set ```.env``` file with environment variables needed
- AIR_VISUAL_API_KEY: the api key of your account at IQAIR
- MONGODB_URI: MongoDB connection string

```bash
# development mode to run Node.js application
$ npm install
$ npm start
```
## Unit Test

```bash
# use this command to run unit test
$ npm run test
$ jest --coverage
```
